## Tavsiyeler:

1. **Profil Fotoğrafı**: Profesyonel, net ve iyi aydınlatılmış bir fotoğraf kullanın
2. **Proje Görselleri**: Projenizin ekran görüntülerini veya temsili görselleri kullanın
3. **Optimizasyon**: Görselleri yüklemeden önce optimize edin (TinyPNG, ImageOptim gibi araçlarla)
4. **Format**: Web için JPEG (fotoğraflar için) veya PNG (logo/icon için) kullanın
